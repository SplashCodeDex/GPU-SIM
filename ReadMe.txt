Note:
Before updating this firmware, ensure that you back up the data in the system and close all open applications. It is also recommended to connect the power cable when it is not a desktop and disconnect all external storage devices.

Firmware upgrade through the Windows utility

Upgrade Procedure
1. Close all open applications, and save your work before the firmware update process.
2. Go to the location where you have downloaded the package.
3. Right-click the file that is named SNT_Dell_SC311_vE104_70002P10 and select Run as administrator to run the firmware (FW) update utility with administrator privilege.
4. Review the License Agreement. If you agree with it, click ACCEPT.
5. A window is displayed where you can select the solid-state drive (SSD) to be updated.
6. Select your drive from the list.
7. Click Start.
8. Click OK to proceed with firmware update.
Do not interrupt the firmware update.
9. A dialog box is displayed when the firmware update is complete. Click OK.
10. Click OK to restart the system.
If you select Cancel, new firmware will be activated until next reboot.
